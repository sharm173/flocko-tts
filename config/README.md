# Configuration Files

Additional requirements files for different environments.

## Files

- **`requirements-local.txt`** - Minimal dependencies for local API testing without model
- **`requirements-docker-minimal.txt`** - Minimal Docker requirements (for testing)

## Main Requirements

The main requirements files are in the root:
- `requirements.txt` - Full requirements for production
- `requirements-docker.txt` - Docker-specific requirements (used by Dockerfile)
